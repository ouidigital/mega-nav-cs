CodeStitch Mega Navigation

A drop-in CodeStitch navigation component that upgrades the stock #cs-navigation pattern with real mega-menu capabilities: multi-column panels, grouped categories, featured promo blocks, and a robust disclosure controller with stack-unwinding Esc.

The top navigation showcases all four column layouts side-by-side:

  • Home — a simple link (no disclosure).
  • Solutions — a 2-column mega panel with two categories.
  • Services — a 3-column mega panel with two categories and a featured promo / CTA block.
  • Resources — a 4-column mega panel with four categories.
  • Company — a standard dropdown, which is the 1-column layout (no .cs-mega needed).
  • Contact — an edge-aligned right-anchored dropdown (.cs-drop-right).

Click any parent item to open its panel.

Try it

  • Resize to <1024px → CS-native drop-from-top drawer with backdrop. All grids collapse to a single column.
  • Press Esc inside a panel to close the deepest level first.
  • Run document.body.classList.add('dark-mode') in devtools.
  • Disable JS — the :focus-within fallback still opens panels.

Iconography library

All icons ship inline in the SVG sprite at the bottom of this file. Reference them with <use href="#icon-name"> — stroke inherits from currentColor, so they recolor with the surrounding text.
